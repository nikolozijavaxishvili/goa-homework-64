# ახსენით რისთვის გამოიყენება შემდეგი თეგები: nav, form, input, label და ატრიბუტები: type, id, for # (რისთვის გამოიყენება nav ტეგი) ნავ თეგი არის HTML ელემენტი წარმოადგენს გვერდის ნაწილს, რომლის მიზანია ნავიგაციის # (რისთვის გამოიყენება form) ფორმ თეგი არის HTML ფორმა გამოიყენება მომხმარებლის შეყვანის შესაგროვებლად. # (რისთვის გამოიყენება ინპუტი) ინპუტ თეგი არის HTML ელემენტი გამოიყენება ვებზე დაფუძნებული ფორმების ინტერაქტიული კონტროლის შესაქმნელად, # (რისთვის გამოიყენება label) ლაბელ ტეგი არის HTML-ში ტეგი მუშაობს შეყვანის ელემენტებთან ერთად. ეს საშუალებას აძლევს მომხმარებლებს დააწკაპუნონ ეტიკეტზე, # (რისთვის გამოიყენება ტიპი) ტიპი ტეგი არის HTML-ში type ატრიბუტი განსაზღვრავს ელემენტთან დაკავშირებული შინაარსის ტიპს # (რისთვის გამოიყენება იდ) რისთვის ვიყენებტ იდს ID ატრიბუტის დანიშნულებაა ერთი ელემენტის იდენტიფიცირება დაკავშირებისას (ფრაგმენტის იდენტიფიკატორის გამოყენებით), სკრიპტირების ან სტილის (CSS-ით) # შექმენით ერთგვერდიანი ვებსაიტი GOA-ს შესახებ, რომელზეც დაამატებთ ნავიგაციას, საიტზე უნდა იყოს მინიმუმ 3 სხვადასხვა სექცია. ამასთან აუცილებლად უნდა იყოს ფორმა, რომლის საშვალებითაც სტუმარი შეძლებს GOA-ში რეგისტრაციას და ჩადობას
